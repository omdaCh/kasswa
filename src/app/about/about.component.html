About This Project

Welcome to the Kasswa project! This open-source e-commerce platform is a personal initiative designed to help me learn and master Angular and Angular SSR while improving my overall development skills.

Technologies Used

  • Front-end: Angular, Angular SSR, Bootstrap, Tailwind CSS, Jasmine, Karma, Cypress, GitLab CI/CD pipelines.
  • Back-end: Express.js, MongoDB, Jest, GitLab CI/CD pipelines.

Future Plans

As I continue to develop this project, I plan to add essential features such as, authentication, admin space for managing items and orders.

Get Involved

This project is open-source, and contributions are welcome! If you're interested in collaborating, feel free to explore the codebase on GitHub "https://github.com/omdaCh/kasswa" and share your feedback or suggestions.
